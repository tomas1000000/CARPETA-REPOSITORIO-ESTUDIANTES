# LA ARQUITECTURA DEL PENSAMIENTO 

### **Máquina diferencial** *(1822- Babbage)*  
Babbage diseñó esta maquina para sistematizar el calculo y la tabulación de funciones polinomicas.  

### **Máquina Analítica** *(1834-1837)*  
- El molino: La CPU o procesador -> dónde se hacen las operaciones.
- El almacen: La memoria -> dónde se guardan los datos.
- Las tarjetas perforadoras: -> la entrada de datos.

### **Telar de Jacquard** *(1801)*  
- Primer sistema de almacenamiento de info binaria.
- El telar usaba un sistema de tarjetas perforadoras para lograr diseños o dibujos complejos y automatizados.
-  **SI HABÍA AGUJERO**: La varilla pasaba, el gancho se activaba y levantaba el hilo (Se puede llamar un 1 o "encendido").
-  **SI NO HABÍA AGUJERO**: La varilla centra la tarjeta, el gancho no se movía y el hilo se quedaba abajo (Podemos llamarlo un 0 o "apagado").
