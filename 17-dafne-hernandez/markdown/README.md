# Formato de TEXTO

Negrita  **-->**  ** Texto ** o __ Texto __  (todo junto)  **-->**  **Hola**  
Cursiva  **-->**  * Texto * o _ Texto _  **-->**  *Holi*  
Negrita y  Cursiva  **-->**  *** Texto ***  **-->**  ***Holis***  
Tachado **-->**   ~~ Texto ~~  **-->** ~~Hello~~  
Escapado de simbolos  **-->**  \# Texto  **-->**  \#Holis
